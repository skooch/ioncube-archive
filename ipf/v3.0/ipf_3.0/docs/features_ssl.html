2.5. SSL Support

Installers built with IPF can be built with SSL support. End-users will then be able to install to web servers accessed via HTTPS.

The cryptographic libraries which enable SSL support are fairly large, so SSL support is provided as an option. If bandwidth usage is a particularly important concern then it might be desirable to make two versions of a given product avilable for download: one version with SSL support and one without.
