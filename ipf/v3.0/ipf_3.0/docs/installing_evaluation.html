3.2. Evaluation Version Key

If you have downloaded an evaluation copy of Package Foundry you will need an evaluation key. This key will have been emailed to you at the email address you provided when downloading the evaluation. The key consists of 8 letters or digits.

When using the Windows IPF GUI you will be prompted to enter your evaluation key the first time you build a product.

When using the command-line version of IPF on Linux or Windows, you will need to use the --eval-key option, followed by the 8 character evaluation key.
