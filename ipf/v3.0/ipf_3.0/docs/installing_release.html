3.1. IPF Licensing

To help protect against piracy the ionCube Package Foundry uses license files which contain machine-specific data including data related to network cards. A license file lic.txt must be placed in the main Package Foundry application directory before products can be successfully built.

To activate the full version of the Package Foundry, first choose License Request from the Help menu. A license request text files will be created in the Package Foundry application directory for each installed network card and a dialog will appear listing the file or files created.

As described in the dialog, one of these files should be emailed to licenses@ioncube.com. A license file lic.txt will be emailed back to you and should be placed in the Package Foundry application directory.
